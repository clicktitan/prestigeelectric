# Prestige Electric Website

Static marketing website for Prestige Electric, a Kansas City metro electrician serving residential and commercial customers.

## What is included

- Clean landing-page design inspired by modern electrician service sites
- Prestige green and charcoal brand styling with an inline SVG wordmark
- SEO-focused metadata, service copy, local service-area content, and JSON-LD schema
- Quote form UI that prepares an email request
- `site-content.json` for services, FAQ items, service areas, and core business details

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production build is generated in `dist/`.
