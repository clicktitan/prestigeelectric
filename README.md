# Prestige Electric

A simple static website starter for Prestige Electric.

## Run locally

From the repository root:

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:5173
```

In Cursor Cloud, open the forwarded/preview URL for port `5173` from the Ports
or Preview panel instead of opening your own computer's localhost directly.

Edit `index.html` and `styles.css`; Vite will reload the page as you make changes.

## Production build

```bash
npm run build
```
